# 📱 Social Media App

A modern and scalable social media application built using Flutter and Firebase with a focus on clean UI, smooth performance, and real-time user interaction. The application is designed with production-level architecture and optimized user experience.

- Story Upload & Feed System  
- Like, Comment & Share Functionality

- Responsive & Modern UI  
- Smooth Animations & Transitions  
- Dark Theme Support  
- Firebase Integration  
- API Ready Architecture

- Error Handling & Loading States
